# Seed Data

Run seeds after migrations:

```bash
npm run seed
```

## What gets seeded

1. **Categories** (`001_categories.sql`) — Engineering, Design, Marketing, etc.
2. **Users & Jobs** (`seed.js`) — Demo admin/user accounts and sample job postings
